# SolEngine

SolEngineはDirectX12を使用したC++ゲームエンジンです｡
アーキタイプ式ECSを主軸に開発しています｡

機能
- 
- アーキタイプECSを利用したゲーム設計
- Blenderを使用したレベルエディタ
- Flyweightパターンに対応したリソースマネージャ
- マルチメッシュのOBJ,GLTFの読み込み､描画
- マルチメッシュモデルへのスキニングアニメーション
